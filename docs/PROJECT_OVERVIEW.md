# Project Overview

## What This Is

A standalone proposal website for the **Arena Holdings Content MVNO** -- a content-led mobile virtual network operator leveraging Arena Holdings' media, music, broadcasting, and events portfolio to create a differentiated telecoms offering in South Africa.

## Purpose

This site serves as:
- A digital proposal document for Arena Holdings decision-makers
- A visual representation of the MVNO strategy, product tiers, and content integration model
- A leave-behind asset for board-level and executive stakeholders
- A demonstration of the digital capability that would underpin the MVNO's customer experience

## Target Audience

- Arena Holdings executive leadership and board
- Arena Holdings divisional heads (News, Music, Broadcasting, Events)
- Potential MNO partners (Cell C, MTN)
- Digital Mobile / MVNE partners
- Investors and strategic advisors

## Technology Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Components:** shadcn UI
- **Deployment:** Vercel
