# Scope

## Pages Included

1. **Home / Hero** -- Strategic overview, headline value proposition
2. **The Opportunity** -- SA MVNO market context, why now, why Arena
3. **Content Arsenal** -- Arena Holdings' full media portfolio mapped to MVNO value
4. **Product Tiers** -- Arena Connect / Arena Plus / Arena Black tier breakdown
5. **Gallo Music & CRBT Strategy** -- Music catalogue, CCA integration, CRBT differentiator
6. **Property Strategy Map** -- Each media property mapped to telecom product and segment
7. **Revenue Model** -- Layered revenue stack, ARPU projections, customer value flywheel
8. **Next Steps** -- Implementation workstreams, timeline, call to action

## Sections Included

- Navigation header with anchor links
- Hero section with key statistics
- Content cards for each Arena division
- Interactive tier comparison (tabs or cards)
- Artist showcase / catalogue highlights
- Data tables for audience reach and ARPU
- Revenue stack visualisation
- Contact / next steps CTA
- Footer

## What Is Excluded

- User authentication / login
- Payment processing
- CMS or admin panel
- Blog or news feed (live content)
- Analytics dashboard
- API integrations with Arena systems
- Multi-language support (English only)
